# 💻 Laptop Price Prediction

A machine learning project that predicts laptop prices based on technical specifications using regression models. Built with a web-based UI using Streamlit.

---

## 🎯 Objective

To help users estimate the price of a laptop by entering key hardware features like processor, RAM, SSD, screen size, etc.

---

## 🛠️ Tech Stack

| Tool         | Purpose                      |
|--------------|------------------------------|
| Python       | Model training & processing  |
| Streamlit    | Web interface                |
| scikit-learn | ML models (Linear Regression, Decision Tree) |
| pandas, numpy| Data manipulation            |
| matplotlib, seaborn | Visualizations        |

---

## 🧠 ML Models Used

- Linear Regression
- Decision Tree Regressor

Evaluated using:
- R² Score
- MAE / RMSE
